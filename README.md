# Project-2-Handwritten-Digit-Recognizer-MNIST-CNN-
✅ Handwritten Digit Recognizer :-Handwritten Digit Recognizer is a deep learning project that uses a Convolutional Neural Network (CNN) to identify digits from 0 to 9 using the MNIST dataset. The model processes normalized image data, extracts features through convolution layers, and achieves high accuracy in digit classification tasks.
